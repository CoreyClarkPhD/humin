Assigned Task

Task 1: Build Plank Contraption

Using MFP create a planks model. Utilize the Resource, Crafter and Output nodes in the material explorer inside of the MFP to create a model that is capable of producing 8 planks per hour. Once the model is finished, submit it so we can use it in our future builds.

Task 2: Build Stick Contraption

Using MFP create a sticks model. Utilize the Resource, Crafter and Output nodes in the material explorer inside of the MFP to create a model that is capable of producing 32 sticks per hour. Once the model is finished, submit it so we can use it in our future builds.

Task 3: Build Diamond Axe Contraption

Using MFP create a diamond axe model. Utilize the Resource, Crafter and Output nodes in the material explorer inside of the MFP to create a model that is capable of producing 2 axe's per hour. Once the model is finished, submit it so we can use it in our future builds.

Task 4: Build Chest Contraption

Using MFP create a chest model. Utilize the Resource, Crafter and Output nodes in the material explorer inside of the MFP to create a model that is capable of producing 10 chest per hour. Once the model is finished, submit it so we can use it in our future builds.

Below are description of Unit Test failures. You can also goto the Bulider Bot (BB) tab to find bookmarked links to the various topics covered within the tutorial

Incorrect Output Node

  • The required Output Node(s) is missing
  • The incorrect Output Node(s) is loaded

Incorrect Craftable Node

  • The required Craftable Node(s) is missing
  • The incorrect Craftable Node(s) is loaded

Incorrect Resource Node

  • The required Resource Node(s) is missing
  • The incorrect Resource Node(s) is loaded

Model Connections Incorrect

  • The model is missing connections between nodes

Model Output Incorrect

  • Missing inputs into model
  • The model is producing too many items
  • The model is producing too few items

Incorrect Custom Node

  • The Custom Node module is missing
